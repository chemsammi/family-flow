Family Flow starter

Files:
- index.html  -> the website
- state.json  -> the editable state/data file

Important:
- This starter is designed for hosted static web use.
- The webpage reads state.json.
- The webpage does NOT directly write to state.json when hosted statically.
- Your future Shortcut will be the writer.

Starter plan:
1. Host both files together.
2. Open index.html in the tablet browser.
3. Use Shortcuts on your phone to update state.json.

Good first Shortcut actions later:
- Reset both girls to currentIndex = 0
- Toggle homework on/off by editing the steps list
- Set mode to flow at 7:45
